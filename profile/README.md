## What we build

Orboto is a modern alternative to mainstream project-management tools for teams that want
**full control over their data**, **deep AI integration**, and a system that
**autonomous coding agents can talk to natively**.

One platform for everything around getting work done:

- 🎯 **Plan** — Tickets, milestones, sprints, Gantt timelines, capacity &amp; budget
- 🚀 **Execute** — Kanban, list, real-time collaboration, time tracking, timesheets
- 📚 **Document** — Nested wiki with Smart Links, PDF export, backlinks
- 🤖 **Automate** — AI assistant, project primer, OQL/JQL search, webhooks, MCP server
- 🔒 **Govern** — PBAC, SSO (OIDC + SAML), audit log, approval workflows
- 🔄 **Migrate** — Import entire projects from any mainstream project-management tool or CSV

## Built for the AI-agent era

Orboto is one of the few project trackers that treats autonomous coding agents
as **first-class users**, not as an afterthought:

- **REST + OpenAPI** — every UI action is a documented API call
- **MCP server** — connect Claude Desktop, Cursor, or Copilot Chat in seconds
- **Tracker adapter** — drop-in surface for coding-agent orchestrators
- **AI Project Primer** — structured, operator-verified context that grounds
  every AI feature in your project's reality
